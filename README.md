## Вопросы и Ответы
1. Зачем нужна строгая типизация в PHP и как она помогает при разработке?

Она заставляет PHP строго проверять типы данных и предотвращает ошибки при передаче неправильных значений, повышая надежность кода.

2. Что такое класс?

Класс - это шаблон объекта.
Его основные компоненты - это:
- свойства (переменные),
- методы (функции),
- конструктор,
- модификаторы доступа.

3. Объясните, что такое полиморфизм и как он может быть реализован в PHP.

Полиморфизм - это возможность работать с разными объектами через общий интерфейс.
В PHP реализуется через:
- интерфейсы,
- наследование,
- переопределение методов.

4. Что такое интерфейс в PHP и как он отличается от абстрактного класса?

Интерфейс - это структура (контракт), задающая список публичных методов, которые обязаны реализовать классы.

У Интерфейса: только сигнатуры методов,   нет свойств,   класс может реализовать несколько    
У Абстрактного класса: может иметь реализацию, свойства разрешены,   наследование только одно
